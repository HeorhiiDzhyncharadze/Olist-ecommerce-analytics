# Olist E-commerce Analytics

This project analyzes the Brazilian Olist e-commerce dataset using SQL and Python.

## Project Goals

- Explore e-commerce data
- Perform data cleaning
- Analyze sales performance
- Build business insights

## Tools

- PostgreSQL
- Python (Pandas, Matplotlib, Seaborn)
- Jupyter Notebook

## Dataset

Dataset source:
https://www.kaggle.com/datasets/olistbr/brazilian-ecommerce

Raw CSV files are not stored in this repository.

## Project Structure

olist-ecommerce-analytics/

data_raw/ - raw csv files (not tracked by git)

sql/ - SQL queries

notebooks/ - Python analysis

docs/ - documentation
