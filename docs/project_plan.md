# Olist Analytics Project Plan

## Goal

Analyze the Olist e-commerce dataset and generate business insights.

---

## Step 1 — Data Understanding

Tables:

- customers
- orders
- order_items
- payments
- products
- sellers
- reviews
- geolocation

Tasks:

- Check row counts
- Check missing values
- Check duplicates
- Understand columns

---

## Step 2 — Data Cleaning

Tasks:

- Fix data types
- Handle missing values
- Remove duplicates

---

## Step 3 — Exploratory Data Analysis (EDA)

Questions:

- How many orders per month?
- Revenue trend
- Top product categories
- Top sellers
- Delivery time analysis

---

## Step 4 — Business Insights

Possible insights:

- Seasonality
- Best categories
- Customer behavior
- Delivery performance

---

## Tools

- PostgreSQL
- Python (Pandas)
- Matplotlib
